# OoT3D Randomizer Azahar Documentation

Describes how to use OoT3D Randomizer on Azahar emulator.

## On Azahar

- Launch OoT3D_Randomizer.3dsx. 
- Navigate the menus to choose the settings for your playthrough, and finish by selecting Generate Randomizer.
- When prompted, select Citra emulator. 
- After the app has finished, open your Azahar "User" folder (specified when first starting the emulator). 
- Navigate to \sdmc\luma\titles\0004000000033500. You should find files called code.ips, exheader.bin, and a folder called romfs. 
- Copy everything from here.
- Go back to your "User" folder, then navigate to \load\mods\0004000000033500 (create the folders if they don't exist).
- Paste the files you copied earlier.
- Run OoT and the title screen should show "Randomizer" below the title name.
